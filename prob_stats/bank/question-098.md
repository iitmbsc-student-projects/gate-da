A repairman services five washing machines each day. For each machine, there is a $50\%$ chance that the repair will involve installing a specific part. Find the probability  that at most one washing machine out of the five will need this part for installation.

- [ ]  $\dfrac{1}{32}$\vspace{1mm}

  
- [ ] $\dfrac{5}{32}$\vspace{1mm}

  
- [ ] $\dfrac{6}{32}$\vspace{1mm}
  

- [ ]  $\dfrac{3}{32}$
 
  
::: {.callout-note title="Hint" collapse=true}




:::

::: {.callout-note title="Answer" collapse=true}

- [ ]  $\dfrac{1}{32}$\vspace{1mm}

  
- [ ] $\dfrac{5}{32}$\vspace{1mm}

  
- [x] $\dfrac{6}{32}$\vspace{1mm}
  

- [ ]  $\dfrac{3}{32}$
 
  
:::
